# Netflix Data Insights Dashboard

## Project Overview
This project is an interactive Tableau dashboard analyzing Netflix movies and TV shows. It provides insights into content trends, ratings, genres, and country-wise distribution.

## Dashboard Highlights
- **Genre Analysis:** Bar chart showing number of shows by genre  
- **Yearly Trend:** Line chart tracking content added over years  
- **Geographical Distribution:** Map of shows by country  
- **Ratings Analysis:** Scatter chart of ratings vs number of shows  
- **Interactive Filters:** Filter data by year, genre, and country  

## Tools & Dataset
- **Tool:** Tableau Desktop / Tableau Public  
- **Dataset:** CSV file included in this repository (`Netflix Dataset.csv`)  

## Files Included
- `Netflix_Dashboard.twbx` → Tableau dashboard file  
- `Netflix Dataset.csv` → Dataset used in the dashboard  
- `README.md` → Documentation  
- `Screenshots/` (optional) → Images of dashboard visuals  

## How to Explore
1. Open `Netflix_Dashboard.twbx` in Tableau.  
2. Use filters to select specific years, genres, or countries.  
3. Hover over chart elements for detailed information.
